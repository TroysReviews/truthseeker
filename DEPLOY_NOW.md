# 📱 Deploy to Android + Streamlit Cloud

## 🚀 Quick Deployment (5 minutes)

### Step 1: Deploy to Streamlit Cloud

1. Go to **https://share.streamlit.io**
2. Click **"New app"**
3. Enter:
   - GitHub repo: `TroysReviews/truthseeker`
   - Branch: `main`
   - Main file: `truthseeker_mobile.py`
4. Click **"Deploy"**

✅ Your app is now live!

---

## 📱 Install on Android (3 steps)

### Chrome Browser
1. Open your Streamlit Cloud URL
2. Tap **⋮** (menu) in top-right
3. Tap **"Install app"**
4. Tap **"Install"**

**Done!** App appears on your home screen.

### Firefox Browser
1. Open your Streamlit Cloud URL
2. Tap **⋯** (menu)
3. Tap **"Install"**
4. Confirm

### Edge Browser
1. Open your Streamlit Cloud URL
2. Tap **"..."** (menu)
3. Tap **"Apps" → "Install this site as an app"**
4. Confirm
